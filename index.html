Introduction

PHP (recursive acronym for PHP: Hypertext Preprocessor) is a widely-used open source general-purpose scripting language that is especially suited for web development and can be embedded into HTML.

Learn the PHP by CLI first, if you understand like define variable, conditional statement, loop statement, data structure. Feel Free to learn make PHP Web Application.

Hello World

To get started with PHP, write your first "Hello World" in your VSCode

$name 'World!'
echo 'Hello' . $name

To run PHP. Save it as index.php then CTRL + ` to open vscode terminal then write "php index.php" on your vscode terminal.

The result will show like this :

php index.php Hello World!

Note! PHP endline with semicolon (;)

Variables

In PHP, a variable starts with the $ sign, followed by the name of the variable

$name called string

$name 'Sandika'

$age called int

$age 20

$money called float

$money 20.4

$isMale called bool

$isMale true

If you run this code, it will not shown the result. Because the variable only store the value in it. To show the variable value use echo function.

$name 'Sandika'

$age 20

$money 20.4

$isMale true

will print Sandika

echo $name

NOTE! PHP automatically associates a data type to the variable, depending on its value. Since the data types are not set in a strict sense, you can do things like adding a string to an integer without causing an error.

Condition

If Statement is set a condition to program what it should do if the statement is fullfiled

$statement false

if ( $statement ) {

... do something here

} elseif ( $statment ) {

... do something here if the first if is not filled

} else {

... do something if all if and elseif statement not filled

}

The value of statement must be true to execute if statement, if false the else statement will executed

Operator

There are some operator in PHP :

  • Logical Operator (basically used in logical statement like if, while)
